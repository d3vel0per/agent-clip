实现冒泡排序的“优雅”在于保持其结构的纯粹，同时利用 Python 的特性提升可读性。下面是一个带有优化标志（Early Exit）的实现：

def bubble_sort(items):
    """An optimized Bubble Sort implementation."""
    n = len(items)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if items[j] > items[j + 1]:
                items[j], items[j + 1] = items[j + 1], items[j]
                swapped = True
        if not swapped:
            break
    return items

Certainly. Let us examine the computational complexity of this procedure:

  • Worst Case ($O(n^2)$): 当输入列表完全逆序时。Occurs when the list is sorted in reverse order.
  • Best Case ($O(n)$): 得益于 swapped 标志，如果输入已排序，算法仅需遍历一次。Thanks to our optimization, a sorted list requires only one pass.

Bubble Sort is remarkably efficient in terms of memory usage:

  • Space Complexity ($O(1)$): 这是一个“原地”（in-place）排序算法。It requires only a constant amount of additional space for variables like swapped and loop indices, regardless of the input size.

This makes it an excellent choice for environments where memory overhead must be strictly controlled.
